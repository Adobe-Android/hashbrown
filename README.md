# hashbrown
You haven't eaten breakfast yet and you're craving hashbrowns but still have to work? This is the application for you! With hashbrown, you can verify your files with various checksum options to ensure you have the correct file and its not corrupted.

## Compatibility
Works with Windows. Microsoft .NET Framework 4.7.2 or later.

## Usage

### Clone the repo
```
git clone git@github.com:Adobe-Android/hashbrown.git
```
### Open the solution (.sln) file in Visual Studio
![Visual Studio solution shown in File Explorer](screenshots/hashbrown-sln.png)

### Run
Click Start in the Visual Studio menu (the green play button)

![Visual Studio menu](screenshots/vs-menu.png)
